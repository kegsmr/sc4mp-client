SC4MP Launcher Readme for Version 0.1.0

Thank you for joining Poppy Multiplayer Regions! This Readme will help you get started with the PMR Launcher. With just a few steps, you can connect to and play in multiplayer regions.

Getting started

Get a PMR account

PMR uses accounts to verify everyone’s saves. So, before you can get started with the client, you will need to join PMR.

Run the launcher

Open ‘PMR Launcher’ to start the launcher. Since this your first time starting the launcher, it will give you a handy welcome message. Close it, and the Region Listings will open:

Connect to a region

Choose a region by selecting it in the listing. Then, click “Connect” to play in it. You will be asked to log into your PMR account:

Once you log in, the launcher will start downloading the region:

Finally, after the region has been downloaded, the launcher will automatically start SimCity 4. The Launcher Map will also open.

Congrats! You have just connected to a Poppy Multiplayer Region. Read on to learn how to claim and save your city.

Playing in a multiplayer region

How claims work

PMR uses a claims system to protect cities from editing and to ensure that they are distributed fairly.

In this system, players can only save cities that they have previously claimed (all other saves are ignored by PMR). To claim an unoccupied tile, open it and save it. Choose wisely: you can only claim one tile per region, and you cannot abandon a claim.

Using the Launcher Map

The Launcher Map is an important tool to use with PMR. It shows which tiles are unoccupied (claimable), occupied, or claimed by you. For instance, this map shows that no tiles have been claimed yet:

Claiming a tile

To claim an unoccupied tile (colored blue in the Launcher Map), just open and save it in SimCity 4. Within a few seconds, the launcher will recognize your save and try to push it to PMR:

If your claim went through, you will see the tile turn green in the Launcher Map. If the claim was unsuccessful, the launcher will alert you and explain why.

Please note that you can only make one claim per region. You cannot abandon claims.

Playing in a claimed city

Once you have claimed your city, you can play in it and save it as usual. As long as the launcher is open and connected to PMR, your saves will be pushed automatically.

Playing in others’ cities

You are more than welcome to open and play in others’ cities. You can even edit and save them locally, but keep in mind that the changes will be ignored by PMR.

Refreshing the region

Although your saves are automatically pushed, the launcher cannot automatically pull others’ new changes due to in-game restrictions. To manually refresh the region, click “Refresh Region” from the Launcher Map and follow the on-screen instructions.

Disconnecting

To disconnect from a region, click “Disconnect.” Make sure that your last save has finished being pushed to PMR before disconnecting.

NEW! Use your own plugins

Although each region comes with a set of plugins, you may find it useful to use some of your personal plugins, such as the Network Addon Mod, with PMR. To do this, simply move your plugins from the regular "SimCity 4\Plugins\" directory to "SimCity 4\Plugins\_PMR\Plugins\".

The moved plugins will still be enabled when you play SimCity 4 without PMR, because the PMR plugins folder is located within the normal plugins folder.

Configuring the launcher

To configure the launcher, click “SC4 Settings…” from the Region Listings. The following window will appear:

PMR working path

When the launcher downloads regions, it writes to a special sandboxed folder. This prevents the launcher from interfering with your single-player regions and plugins. By default, the launcher will write to a directory named “_PMR” from the SimCity 4 directory in Documents.

If you want the launcher to write its files to another directory, specify it in this field. Do NOT set this path to your base SimCity 4 directory (“Documents\SimCity 4\”)! If you do this, the launcher will overwrite your regular regions and plugins. Unless you have a specific reason to do so, it’s best to keep the default path.

The default value is "C:\Users\[USERNAME]\Documents\SimCity 4\_PMR\"

Custom resolution

In order to keep the Launcher Map in view, the launcher will always start SimCity 4 in windowed mode. However, you can specify a custom resolution for the window.

The default value is 1280 × 800 pixels.

Path to SimCity 4.exe

When attempting to launch SimCity 4.exe, the launcher will look for it in the standard installation locations. If the launcher cannot find your copy of the executable, you can provide a custom path to it.

You only need to set this if the launcher asks you to.

The default value is blank.

Resetting the launcher

To reset the configuration file of the launcher, simply empty the contents of “config.ini” in the resources directory (but keep the file).

Recovering from PMR interruptions

The PMR Launcher has been designed to recover from most common errors. However, if PMR experiences a temporary outage, or the launcher unexpectedly quits, you may need to recover your latest save.

There is no automatic recovery system, but you can manually recover your unpushed saves by copying them from the PMR Regions directory to another folder. Then, when you reconnect to the region, you can copy it back. You may need to open and re-save your recovered city for the launcher to push it.
